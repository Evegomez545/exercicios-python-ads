# 🚀 Meus Estudos de Python (ADS)

## 🎓 Sobre o Curso
O primeiro módulo (Mundo 1) foca nos fundamentos da linguagem e na lógica de programação, preparando a base para estruturas mais complexas.

## 📂 O que foi praticado:
* **Tratamento de Dados:** Tipos primitivos, entrada e saída de dados.
* **Operadores Aritméticos:** Cálculos, precedência e funções matemáticas (módulo `math`).
* **Manipulação de Texto:** Fatiamento de strings, análise de frases e transformações.
* **Estruturas de Controle:** Uso de condições simples e compostas (`if/else`).
* **Desafios de Lógica:** Exercícios envolvendo situações do dia a dia (cálculo de descontos, multas, triângulos, etc.).

## 🛠️ Tecnologias
* **Linguagem:** Python 3.13
* **IDE:** PyCharm
* **Ferramenta:** Git & GitHub
